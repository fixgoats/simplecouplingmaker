# Leiðbeiningar

keyra:
```
cmake -B build -DCMAKE_BUILD_TYPE=Release
cmake --build build
```
búa til skrá sem heitir `points.txt` í `build möppunni` með punktunum sem maður vill tengja saman.
Keyra:
```
./hamilton
```
Það ætti að spýta út textaskrá `hamiltonian.txt` með stuðlum hamilton virkjans.
Þessi útgáfa setur allar kúplanir jafnt og 1 með engri on-site orku.
